# Trader Performance vs Market Sentiment

##  Project Overview
This project analyzes trader performance and behavior in relation to market sentiment using historical trading data and the Fear & Greed Index.

The goal is to understand how trader profitability, activity, and consistency change under different market conditions and to generate actionable trading insights.

---

##  Dataset Used
1. historical_data.csv  
2. fear_greed_index.csv  

---

##  Tools & Technologies
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  
- Streamlit  

---

##  Key Tasks Performed
- Data cleaning and preprocessing  
- Missing value and duplicate handling  
- Timestamp conversion and alignment  
- Feature engineering  
- Exploratory Data Analysis (EDA)  
- Trader segmentation  
- Clustering using KMeans  
- Dashboard creation using Streamlit  

---

##  Outputs
- processed_data.csv  
- Jupyter Notebook with full analysis  
- Streamlit dashboard  
- Charts folder with visualizations  

---

##  How to Run
1. Install required libraries 
2. Run notebook to generate processed_data.csv  
3. Run Streamlit app
